# PORTFÓLIO PESSOAL BY ISAAC E MARIA LUIZA

Este projeto consiste no desenvolvimento de um **Portfólio Pessoal Responsivo** para a disciplina de **Programação para a Internet I** . O objetivo é consolidar conhecimentos em estruturação e estilização web seguindo padrões profissionais .

## 📌 Sobre o Projeto
A proposta deste trabalho é criar uma interface que apresente informações pessoais, habilidades, projetos e experiências . Estruturamos o site com uma **Landing Page central**:
* **Página de Boas-Vindas:** Uma recepção inicial simulando a página profissional de um estudante/desenvolvedor .
* **Apresentação da Dupla:** Uma seção dedicada com fotos e uma breve explicação sobre Isaac e Maria Luiza
* **Navegação Individual:** Caso o visitante queira adentrar no portfólio específico de um de nós, basta clicar para ser encaminhado a uma nova página detalhada.

## 👥 Integrantes e Tema
* **Integrantes:** Isaac e Maria Luiza.
* **Tema Escolhido:** Estudantes de Tecnologia / Desenvolvedores .

## 🛠️ Tecnologias e Restrições
O projeto foi desenvolvido seguindo restrições técnicas rigorosas:
* **HTML5 e CSS3 Puros:** Uso obrigatório de código nativo sem auxílio de frameworks.
* **Sem Bootstrap ou Tailwind:** Proibição do uso de bibliotecas externas de estilização.
* **Semântica:** Uso correto de tags como `header`, `main`, `section`, `article` e `footer` .

## 📱 Responsividade e Design
O layout foi planejado para se adaptar a diferentes tamanhos de tela :
* Uso de **Media Queries** para adaptação mobile .
* Cards empilhados em telas menores para melhor leitura .
* Imagens e elementos visuais totalmente responsivos .

## 🚀 Desafios Encontrados
* Criar um sistema de navegação entre múltiplas páginas mantendo a organização dos arquivos .
* Garantir que o formulário de contato e os links internos funcionem perfeitamente .
* Aplicar estilos visuais avançados (hover, cores e fontes) usando apenas CSS puro .
